# 📊 Loan Default Risk Analysis

## 📌 Project Overview
This project analyzes loan data to identify factors that lead to loan default and build a predictive model.

## 🎯 Objectives
- Identify key factors influencing loan default
- Analyze borrower risk patterns
- Build a predictive model
- Create a Power BI dashboard

## 📂 Dataset
Dataset used: Lending Club Loan Dataset  
Link: https://www.kaggle.com/datasets/wordsforthewise/lending-club

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Scikit-learn
- Power BI
